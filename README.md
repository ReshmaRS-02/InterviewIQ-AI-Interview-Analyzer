# InterviewIQ - AI Interview Emotion Analyzer

InterviewIQ is an AI-powered system that analyzes facial emotions during mock interviews and estimates candidate confidence levels.

## Features

- Real-time emotion detection
- Confidence estimation
- Webcam-based analysis
- Streamlit interactive dashboard

## Technologies Used

- Python
- DeepFace
- OpenCV
- Streamlit

## How to Run

```bash
pip install -r requirements.txt
streamlit run app/dashboard.py
